# Creo Elements

This was an assignment project where I took an existing Lovable-generated website (originally built with TypeScript, TanStack Start, and Framer Motion) and converted it into my own preferred stack, while keeping the original design intact.

## Stack Used

- React (Vite)
- JavaScript
- React Router DOM
- Tailwind CSS
- GSAP (for animations)
- React Icons

## Running the Project

```bash
npm install
npm run dev
```

That's it.
